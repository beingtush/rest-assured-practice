                    RECOMMENDED APPROACH (EASIEST)
================================================================================

STEP 1: Install IntelliJ IDEA Community Edition (FREE)
   
   Download from: https://www.jetbrains.com/idea/download/
   
   Why IntelliJ?
   - Has Maven built-in (no separate installation needed)
   - Automatic dependency download
   - Easy test execution with right-click
   - Great for beginners

STEP 2: Open Project in IntelliJ

   1. Launch IntelliJ IDEA
   2. Click "Open" on welcome screen
   3. Navigate to this folder: "RestAssured Practice"
   4. Click "OK"
   5. Wait 2-5 minutes for IntelliJ to download dependencies
      (Watch the bottom status bar)

STEP 3: Run Your First Test

   1. Navigate to: 
      src/test/java/com/restassured/practice/tests/BasicGetRequestTest.java
   
   2. Right-click on the file name
   
   3. Select "Run 'BasicGetRequestTest'"
   
   4. See results in the bottom panel!

STEP 4: Run All Tests

   1. Right-click on "testng.xml" file in project root
   2. Select "Run testng.xml"
   3. All 10 test classes will execute

================================================================================
                    ALTERNATIVE: INSTALL MAVEN
================================================================================

If you prefer command line or want to learn Maven:

STEP 1: Download Maven
   https://maven.apache.org/download.cgi
   (Download the Binary zip archive)

STEP 2: Extract Maven
   Extract to: C:\Program Files\Apache\maven

STEP 3: Set Environment Variables
   
   Open PowerShell as Administrator and run:
   
   setx MAVEN_HOME "C:\Program Files\Apache\maven" /M
   setx PATH "%PATH%;%MAVEN_HOME%\bin" /M

STEP 4: Verify Installation
   
   Close and reopen PowerShell, then run:
   mvn --version

STEP 5: Build and Run Tests
   
   cd "RestAssured Practice"
   mvn clean install
   mvn test
